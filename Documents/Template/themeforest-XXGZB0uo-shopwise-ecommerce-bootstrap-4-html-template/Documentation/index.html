Configuration and Installation

Shopwise-HTML-Package zip: Main zip file HTML and Documentation Included

Shopwise-HTML-Package.zip: Shopwise-HTML-Package.zip file extract and main HTML template and documentation folder are included.

(1) Shopwise-HTML and (2) Documentation folder

Open the "Shopwise-HTML" folder and run index.html in your browser

  1. Open the Shopwise-HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • /assets/bootstrap - Bootstrap Stylesheets Folder
    • /assets/css - Extra Stylesheets Folder
    • /assets/fonts - Fonts Folder
    • /assets/images - Images Folder
    • /assets/js - Javacripts Folder
    • /assets/owlcarousel - owlcarousel Stylesheet File
    • /index.html - Index File/Homepage
    The other files can be used according to your preferences.
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

FAQ


How to remove the top menu sticky as scrolling?

  • Open Up your HTML file 'Shopwise-HTML/' file location. (For Example, 'index.html'
  • Then Search For the Comment '<!-- START HEADER -->'
  • show this line - <header class="header_wrap">
  • add class name - <header class="header_wrap no-sticky"> (add class name 'no-sticky')

Where i can define my e-mail account to receive the messages from contact form?

  • Open Up your HTML file 'Shopwise-HTML' file location. (For Example, 'Shopwise-HTML/contact.php'
  • show this line number 11 - $mailTo = 'email@example.com'
  • 'email@example.com' enter your email account to receive the messages from contact form
